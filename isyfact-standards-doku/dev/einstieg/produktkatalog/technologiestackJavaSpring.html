Technologiestack: Java/Spring

Diese Seite ist ein Teil der IsyFact-Standards. Alle Inhalte der Seite, insbesondere Texte und Grafiken, sind urheberrechtlich geschützt. Alle urheberrechtlichen Nutzungs- und Verwertungsrechte liegen beim Bundesverwaltungsamt.

Die Nutzung ist unter den Lizenzbedingungen der Creative Commons Namensnennung 4.0 International gestattet.

Die hier aufgeführten Produkte dienen der Entwicklung und dem Test von Anwendungen mit dem Technologiestack Java und Spring (Boot).

Spring Boot Managed Dependencies

Das Framework Spring Boot legt selbst zahlreiche Versionen von Drittanbieter-Bibliotheken fest. Wann immer ein Produkt als Version dieses Icon () enthält, wird die durch Spring Boot festgelegte Version verwendet.

Produkte und ihre Versionen können in der Liste der Spring Boot Managed Dependencies nachgeschlagen werden.
