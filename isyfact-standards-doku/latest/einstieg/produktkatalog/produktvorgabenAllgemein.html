Spring Boot Managed Dependencies

Das Framework Spring Boot legt selbst zahlreiche Versionen von Drittanbieter-Bibliotheken fest. Wann immer ein Produkt als Version dieses Icon () enthält, wird die durch Spring Boot festgelegte Version verwendet.

Produkte und ihre Versionen können in der Liste der Spring Boot Managed Dependencies nachgeschlagen werden.
